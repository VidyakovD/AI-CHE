Telegram

Бот отвечает в личных сообщениях и группах

Где взять Telegram Bot Token?
  1. Откройте Telegram, найдите @BotFather
  2. Отправьте команду /newbot
  3. Введите название бота (как будет видно клиентам)
  4. Введите username (должен заканчиваться на _bot, например aiche_help_bot)
  5. BotFather пришлёт токен вида 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11 — скопируйте сюда

Webhook установится автоматически при сохранении.

ВКонтакте

Отвечает на сообщения в сообществе

Где взять Access Token и ID группы ВКонтакте?
  1. Откройте vk.com/groups → выберите свою группу
  2. В группе: Управление → Работа с API → Ключи доступа
  3. Нажмите «Создать ключ», отметьте права: сообщения сообщества + фото (опц.)
  4. Скопируйте Access Token сюда
  5. ID группы: в адресе vk.com/club123456789 или в Управление → Адрес группы
  6. Включите Long Poll API в настройках API: Callback API → версия 5.131 → подтвердите Callback URL

Confirmation-код подтянется автоматически при сохранении бота.

Авито

Отвечает в мессенджере Авито

Где взять ключи Авито?
  1. Зайдите в Кабинет разработчика Авито (нужен бизнес-аккаунт)
  2. Подайте заявку на доступ к API Messenger (рассмотрение 1–3 дня)
  3. После одобрения: Мои приложения → Создать приложение
  4. Скопируйте Client ID и Client Secret сюда
  5. User ID — ваш числовой ID, посмотрите в URL профиля: avito.ru/user/abc123def
  6. В настройках приложения добавьте webhook URL — он покажется здесь после сохранения

⚠ Авито требует ручной проверки заявки. Бесплатный тариф доступен, но с лимитами.

MAX

Российский мессенджер max.ru

Где взять Bot Token MAX?
  1. Скачайте MAX (российский мессенджер от VK Group)
  2. В поиске найдите бота @MasterBot
  3. Отправьте команду /newbot
  4. Введите название и username бота (как в Telegram)
  5. MasterBot пришлёт длинный токен — скопируйте сюда

Webhook регистрируется автоматически при сохранении. Проверка через dev.max.ru.

Виджет на сайт

Чат-окно в правом нижнем углу

Как встроить виджет на свой сайт?
  1. Включите чекбокс выше → сохраните бота → код появится
  2. Скопируйте <script> снипет
  3. Вставьте перед </body> в HTML каждой страницы
  4. Для безопасности добавьте свой домен в «Разрешённые источники» (защита от встраивания на чужих сайтах)

1. Общие положения

1.1. Настоящее Соглашение регулирует порядок использования платформы «AI Студия Че» (далее — «Сервис»).

1.2. Использование Сервиса возможно только после принятия условий настоящего Соглашения.

1.3. Администрация вправе вносить изменения. Новая редакция вступает в силу с момента публикации.

2. Регистрация и аккаунт

2.1. Для использования Сервиса необходимо создать аккаунт, указав действительный email и пароль.

2.2. Пользователь несёт ответственность за сохранность учётных данных.

2.3. Администрация вправе заблокировать аккаунт при нарушении условий Соглашения.

3. Использование Сервиса

3.1. Сервис предоставляет доступ к AI-ассистентам, генераторам изображений, видео и другим инструментам.

3.2. Запрещается использовать Сервис для создания незаконного или вредоносного контента.

3.3. Администрация не несёт ответственности за результаты, сгенерированные AI-моделями.

4. Оплата

4.1. Расчёты ведутся в российских рублях с точностью до копейки. Подписочной модели нет.

4.2. Стоимость каждой операции рассчитывается автоматически по реальным токенам выбранной модели и курсу валют.

4.3. Оплата через ЮKassa. Кассовые чеки направляются на email (ФЗ-54).

4.4. Возврат неиспользованной части баланса возможен в течение 14 дней с момента последнего пополнения. Бонусные начисления возврату не подлежат.

5. Персональные данные

5.1. Обработка данных согласно ФЗ-152 «О персональных данных».

5.2. Данные не передаются третьим лицам, кроме случаев по законодательству РФ.

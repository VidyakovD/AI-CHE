# Как подключить чат-бота за 2 минуты

Бот сразу заработает, как только вы вставите токен и нажмёте «Сохранить» —
webhook поднимется автоматически, статус станет «Активен».

---

## 🟦 Telegram (самый быстрый способ — 1 минута)

1. В Telegram откройте [@BotFather](https://t.me/BotFather)
2. `/newbot` → имя бота → @username (должен заканчиваться на `bot`)
3. Скопируйте токен вида `1234567890:AAH...`
4. На [aiche.ru/chatbots.html](https://aiche.ru/chatbots.html):
   - **«+ Новый бот»** → название → выберите модель (Claude / GPT-4o)
   - Вставьте **TG-токен** → **Сохранить**
5. Готово — пишите боту в Telegram

> Если что-то не так — нажмите **«▶ Запуск»** на карточке бота

## 🔵 ВКонтакте (3 минуты)

1. Создайте сообщество: [vk.com/groups → создать](https://vk.com/groups?act=create)
2. **Управление → Работа с API → Длинный ключ доступа**
   → Получить → разрешения: messages, manage
3. **Управление → Работа с API → Callback API**:
   - URL: скопируйте из карточки бота на aiche.ru (после первого сохранения)
   - Тип события: **Входящее сообщение**
4. На [aiche.ru/chatbots.html](https://aiche.ru/chatbots.html):
   - VK-токен → ID группы → **Сохранить**
5. Включите сообщения сообщества: Управление → Сообщения → **Включены**

---

## 🟢 Авито Messenger (5 минут — нужна модерация Авито)

1. Подайте заявку на API: [developers.avito.ru](https://developers.avito.ru)
2. После одобрения — получите **Client ID + Client Secret**
3. На [aiche.ru/chatbots.html](https://aiche.ru/chatbots.html):
   - Client ID → Client Secret → User ID (ваш ID на Авито) → **Сохранить**
4. Webhook URL появится в карточке — пришлите его в support@avito.ru
   с просьбой подписать на события `message`

---

## 🌐 Виджет на сайт

1. На [aiche.ru/chatbots.html](https://aiche.ru/chatbots.html):
   - Включите чек-бокс **«Виджет на сайте»** → **Сохранить**
2. Скопируйте `<script src="https://aiche.ru/widget/<id>.js"></script>`
3. Вставьте перед `</body>` на ваш сайт

## ❓ Бот не отвечает — что проверить

| Симптом | Решение |
|---|---|
| TG-бот пишет «Сервис временно недоступен» | Кончились CH на балансе или заблокирован OpenAI-ключ — проверьте в кабинете |
| TG-бот не реагирует вообще | Webhook слетел — нажмите **«▶ Запуск»** или сохраните токен заново |
| VK не отвечает | В сообществе **отключены сообщения** или CALLBACK API не включён на «Входящее сообщение» |
| Все каналы молчат | Откройте **/admin.html → Логи** или напишите в поддержку |

---

## Тарификация

- TG / VK / Авито / Виджет — **бесплатно**, плата только за ответы AI (по реальным токенам модели)
- Стоимость одного ответа Claude Sonnet ≈ **3–10 CH** (зависит от длины)
- См. **Лимиты** в карточке бота — можно ограничить max ответов в день

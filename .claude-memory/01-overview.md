# 01. Обзор проекта

## Что это

**AI Студия Че** — веб-платформа мультипровайдерного AI-ассистента для малого бизнеса в России.

Домен: https://aiche.ru
Владелец: Денис Видяков (vidyakov@obsidian.ai)
Репо: https://github.com/VidyakovD/AI-CHE

## Ключевые модули (что видит юзер)

1. **Чат с AI** (index.html) — выбор модели GPT/Claude/Gemini/Perplexity/Grok + картинки DALL-E + видео Kling/Veo.
2. **Чат-боты** (chatbots.html) — развернуть бота в TG/VK/Авито/виджет на сайт.
3. **ИИ-агенты** (agents.html) — готовые AI-специалисты (юрист, SMM, копирайтер, аналитик тендеров и т.д.) + конструктор воркфлоу.
4. **Готовые решения** (Solutions) — пошаговые AI-сценарии («Составь SWOT», «Напиши КП», …).
5. **Сайты** (sites.html) — AI собирает landing через диалог.
6. **Презентации/КП** (presentations.html) — AI генерирует КП и презентации.
7. **База знаний** — загрузка PDF/DOCX → RAG-ответы.
8. **Админка** (admin.html) — управление юзерами, ключами, ценами, промокодами, audit-log.

## Стек

- **Backend:** Python 3.x, FastAPI, uvicorn (2 workers на проде), SQLAlchemy 2.0
- **БД:** SQLite (`chat.db`) + lightweight-миграции (без Alembic)
- **Frontend:** Чистый HTML/CSS/JS (SPA без фреймворка), темная тема, Tailwind CDN
- **AI провайдеры:** OpenAI (прямой), Anthropic (через прокси awstore.cloud), Google Gemini, xAI Grok, Perplexity, OpenAI DALL-E, Kling, Google Veo
- **Авторизация:** JWT (HS256) через python-jose, bcrypt через passlib, OAuth Google/VK (закодено, ждёт ключи)
- **Платежи:** ЮKassa (yookassa SDK)
- **Hosting:** своя VPS (Ubuntu, 194.104.9.219), Nginx + Let's Encrypt, systemd

## Внутренняя валюта

**CH** (Che). Подробно в `04-billing.md`.

Розничная цена: **1 CH = 0.10 ₽** (хранится в `pricing_settings.ch_to_rub`).

## Бизнес-модель

- Подписки (Старт/Про/Ультра) — дают больше CH за рубль + приоритет
- Разовые пакеты CH
- Реферальная программа (10% от оплаты + бонус за регистрацию друга)
- Промокоды
- Каждый AI-запрос списывает CH по per-token тарифу + фикс-плата за модули

## Ключевые особенности

1. **Один владелец = много клиентов.** Бизнес-юзер регистрируется на aiche.ru, разворачивает бота в своём TG/VK → его клиенты пишут боту → списания идут с баланса владельца.
2. **Приоритет — UX для не-технарей.** Юзеры не разработчики — нужны простые визард-формы, понятные кнопки, подсказки «что нажать». Не гиковские UI.
3. **Много AI-моделей.** Юзер выбирает под задачу: Claude для длинных текстов, GPT-4o для общего, Perplexity для свежей инфы, DALL-E/Kling для картинок/видео.
4. **Русский контент.** Интерфейс только на русском. Все промпты — на русском. Документы/оферта — РФ-законодательство.
5. **Чёрная дыра API-ключей.** Из-за санкций OpenAI/Anthropic в России работают через прокси. Периодически отваливаются. `ANTHROPIC_BASE_URL=https://api.awstore.cloud`.

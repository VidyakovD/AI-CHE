### 2. Google OAuth (Client ID + Client Secret) — для входа через Google
**Зачем:** добавить кнопку "Войти через Google" на главную.
**Где получить:**
1. https://console.cloud.google.com → новый проект
2. APIs & Services → OAuth consent screen (External, scope: email, profile, openid)
3. Credentials → OAuth client ID → Web application
4. **Authorized redirect URI:** `https://aiche.ru/auth/oauth/google/callback`
5. Скопировать **Client ID** и **Client Secret**

**Env переменные на сервере:**
```
GOOGLE_CLIENT_ID=...apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=GOCSPX-...
```

### 3. VK OAuth (Client ID + Secret) — для входа через ВКонтакте
**Зачем:** кнопка "Войти через VK".
**Где получить:**
1. https://dev.vk.com → создать приложение → **Веб**
2. Настройки приложения:
   - **Авторизованный redirect URI:** `https://aiche.ru/auth/oauth/vk/callback`
   - **Базовый домен:** `aiche.ru`
   - Статус: включено и видно всем
3. Скопировать **ID приложения** (client_id) и **Защищённый ключ** (client_secret)

**Env переменные:**
```
VK_CLIENT_ID=12345678
VK_CLIENT_SECRET=abc...
```
